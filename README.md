# Plantilla de Página Web para Empresas o Freelance (Flask)

## Tecnologías utilizadas en el desarrollo:

- Bootstrap 4.
- FontAwesome.
- JavaScript.
- Flask (Python web framework para envío de email contacto).
- HTML + CSS + Jinja2 templates

## Cómo ejecutar:

1. Asegúrate de tener Python instalado.
2. Instala las dependencias: `pip install flask`
3. Ejecuta la aplicación: `python app.py`

## Configuración del email:

En `app.py`, configura los detalles del SMTP para el envío de emails en la función `send_contact`.
